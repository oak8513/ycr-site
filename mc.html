Maintenance Check Up

Hard Drive Health

Monitoring sector health and temperatures to prevent drive failure.

Hard Drive Speeds

Checking read and write speeds in MBs per second to ensure performance.

RAM Quantity

Assessing if the system has sufficient RAM for optimal performance.

3rd Party Updates

Ensuring all 3rd party software is updated to mitigate security vulnerabilities.

Windows Updates

Verifying the installation of the latest Windows updates for security and stability.

Firmware/Bios Updates

Checking if firmware and BIOS are up-to-date to ensure hardware compatibility and security.

Security Status

Evaluating the overall security measures implemented across the system.

Internet Speed Check

Measuring current internet speeds to identify potential network issues.

Managed Service Plan

Reviewing the current managed service plan to ensure all devices are covered.

Clean Fans and Case (Desktop Only)

Verifying that desktop fans and case are clean to prevent overheating.

Malicious Browser Extensions

Identifying and removing potentially harmful browser extensions.

PUPS

Scanning for and removing unwanted programs.

SFC Scan

Running System File Checker (SFC) to repair corrupted system files.

Reliability History Check

Reviewing system reliability history to identify and resolve recurring issues.

Next Gen Antivirus

Ensuring a next-generation antivirus is installed and up-to-date for advanced threat protection.

Remote Session Blocking

Ensuring valid support software is not used maliciously.

CPU Usage

Reviewing CPU usage and temperatures.

GPU Usage

Reviewing GPU usage and temperatures.

Disk Usage

Reviewing Disk usage and temperatures.
